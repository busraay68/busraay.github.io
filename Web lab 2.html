Verilen benzer bir duzenle basit bir web sayfasi oluşturun. CSS'in temel özelliklerini (display, width, height, margin, padding, border, selector) uygulayın. Sadece HTML5 + CSS3 kullanin, hazır framework kullanmak yasak (Bootstrap, Tailwind vb.).

İçerik

Bu alanda örnek icerik metinleri bulunmaktadir. Css ozellikleriyle bu metinler arasında boşluklar (line-height) ve hizalamalar (text-align) düzenlelleririz.

Önce CSS ile kutuları / alanları yan yana getirmeyi, ardindan alt alta yerleştirmeyi öğrendik. Şimdi ise CSS seçicilerini kullanarak grup ve hiyerarşik seçimler yapmayı göreceğiz.

Yan Menü / Bilgi

Bu alan yan menü veya bilgi kismidir, CSS ile genişlik (width), yükseklik (height), kenar boşlukları (margin/padding) ve kenarlik (border) özelliklerini kullanarak düzenleyebiliriz.

Öğrendiklerimizi hiyerarşik ve grup seçicilerle uygulayıp farkli bolümlere stil verebiliriz.
