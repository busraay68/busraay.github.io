# Web Programlama Ödev Portfolyosu

Bu çalışma alanı, Web Programlama dersi süresince hazırladığım ödevleri ve teknik projeleri bir araya getirmektedir.

## Kullanılan Teknolojiler ve Yöntemler
Projeler geliştirilirken aşağıdaki araçlara ve standartlara odaklanılmıştır:

*   **HTML5:** Semantik yapı ve içerik yönetimi.
*   **CSS Flexbox:** Modern ve esnek sayfa düzenleri.
*   **Responsive Design:** Mobil ve masaüstü cihazlara tam uyumluluk.
*   **GitHub Pages:** Hazırlanan çalışmaların canlı ortamda sunumu.

##  Öne Çıkan Çalışmalar
*   **Kişisel Portfolyo Sitesi:** Mühendislik projelerimi ve yetkinliklerimi sergilediğim kapsamlı web sitesi.
*   **Arayüz Tasarımları:** Modern CSS teknikleri ile oluşturulmuş sayfa yapıları.
